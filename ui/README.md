# Database Browser / Admin UI

Lightweight web interface for scientists to browse, query, and manage the Mandarte database without writing SQL.

## Status

🔴 **Not yet started.**

## Planned features

- Browse all working tables (birds, survival, breed) with search and filter
- Inline field documentation (hover over a column to see what it means)
- Pre-built queries for common questions ("show all nests for bird X", "breeding history on territory 14")
- Admin correction workflow with required reason field and audit logging
- CSV/Excel export with configurable missing value format ('.' for R, blank for Excel)
- Read-only access for collaborators
